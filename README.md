# Smart-Security-Camera-System-for-Supermarkets-using-BLIP-and-WebSockets
  A smart security camera system for supermarkets using BLIP and WebSockets. It detects suspicious activities, alerts operators, and stores key event images. Supports multiple camera streams with minimal delay. Easily integrates new cameras via WebSockets. Built with Python, Flask, Flask-SocketIO, OpenCV, and Docker.
